# Google Play Store Analytics Internship Project

## 📌 Project Overview

This project was developed as part of the **Elevance Skills Data Analytics Internship**.

The objective of this project is to analyze Google Play Store applications using Python by implementing six analytical tasks on the provided dataset. The project demonstrates data preprocessing, visualization, interactive dashboards, and business insights.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Plotly
* Matplotlib
* VS Code
* Jupyter Notebook

---

## 📂 Dataset

The project uses:

* Play Store Data.csv
* User Reviews.csv

---

## 📊 Internship Tasks Completed

* Task 1 – Interactive Bubble Chart
* Task 2 – Choropleth Map
* Task 3 – Time Series Line Chart
* Task 4 – Stacked Area Chart
* Task 5 – Grouped Bar Chart
* Task 6 – Interactive Dual-Axis Chart

---

## ✨ Key Features

* Data Cleaning & Preprocessing
* Revenue Calculation for Paid Apps
* Interactive Plotly Visualizations
* Time-Based Dashboard Visibility
* Category Filtering
* KPI Analysis

---

## ▶️ How to Run

1. Download or clone this repository.
2. Open the notebook in VS Code or Jupyter Notebook.
3. Install the required Python libraries.
4. Run the notebook from top to bottom.
